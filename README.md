NAME
====

API::Discogs - Provide basic API to Discogs

SYNOPSIS
========

```raku
use API::Discogs;
```

DESCRIPTION
===========

API::Discogs provides a Raku library with access to the [Discogs](https://discogs.com) data and functions.
